# Quotes App

A free, interactive Quotes App website built with **HTML, CSS, and JavaScript**.

## Features
- Categories: Motivation, Life, Love, Islamic
- Search quotes
- Dark/Light mode
- Copy & Share quotes
- Random background color changes
- Smooth animations
- Mobile-friendly

## How to Use
1. Open `index.html` in your browser or access the live site on GitHub Pages.
2. Select a category from the dropdown.
3. Click "New Quote" to get a random quote.
4. Use the buttons to Copy, Share, or toggle Dark/Light mode.
5. Search for quotes using the search bar.
